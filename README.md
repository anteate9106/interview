# 청년들 입사지원 시스템

입사지원서 작성, 평가자 평가, 관리자 취합을 위한 통합 웹 기반 시스템입니다.

## 📁 시스템 구성

### 1. 지원자 페이지 (`apply.html`)
- 신규 지원서 작성
- 로그인 및 지원서 수정
- 임시 저장 기능

### 2. 평가자 페이지 (`evaluator.html`) ⭐ NEW
- 평가자 로그인
- 지원자 평가 (4개 항목, 각 25점)
- 개별 평가 저장

### 3. 관리자 페이지 (`index.html`)
- 전체 지원자 조회
- 모든 평가자의 평가 취합
- 평균 점수 및 상세 평가 내역 확인

## 🔄 시스템 워크플로우

```
[지원자] apply.html
    ↓ 지원서 작성 및 제출
LocalStorage (applicants)
    ↓
[평가자 1] evaluator.html (evaluator1)
    ↓ 개별 평가
applicant.evaluations[0] 저장
    ↓
[평가자 2] evaluator.html (evaluator2)
    ↓ 개별 평가
applicant.evaluations[1] 저장
    ↓
[평가자 3] evaluator.html (evaluator3)
    ↓ 개별 평가
applicant.evaluations[2] 저장
    ↓
[관리자] index.html (admin)
    ↓ 평가 취합
평균 점수 및 상세 내역 표시
```

## 🎨 지원자 페이지 (apply.html)

### 입력 항목

#### 기본 정보
- 이름, 생년월일, 이메일, 연락처
- 지원 지점 (7개 지점)
  - 영등포점, 수원시청점, 천안아산점, 부산점, 수원인계점, 고양일산점, 청주점
- 지원 직무 (4개 팀)
  - TAX팀, 원천팀, 결산팀, 경리팀
- 비밀번호 (8자 이상)
- 주소, 학력사항, 자격 및 교육사항, 경력사항

#### 작성 항목
1. **자기소개서** - 800자 이내
2. **경력기술서** - 500자 이내
3. **지원동기** - 500자 이내
4. **입사 후 포부** - 500자 이내

### 주요 기능
- ✅ 회원가입 & 로그인
- ✅ 💾 임시 저장 (수동/자동)
- ✅ 실시간 글자 수 카운트
- ✅ 전화번호 자동 포맷팅
- ✅ 지원서 수정 (평가 전)
- ✅ 평가 후 수정 차단

## 📊 평가자 페이지 (evaluator.html)

### UI 구조 - 3분할 레이아웃
1. **왼쪽**: 지원자 목록 (평가 상태 필터)
2. **중간**: 지원서 전체 내용
3. **오른쪽**: 평가 입력 폼

### 평가 항목 (각 25점, 총 100점)
1. **내용충실도** - 지원서 완성도, 구체성, 진실성
2. **경력 및 교육사항** - 경력 적합성, 교육 배경
3. **조직적합성** - 조직 문화 적합성, 팀워크
4. **직무적합성** - 직무 이해도, 필요 역량

### 주요 기능
- ✅ 평가자별 개별 로그인
- ✅ 드롭다운 점수 선택 (25~0점)
- ✅ 평가 코멘트 작성
- ✅ 총점 자동 계산
- ✅ 평가 저장 및 수정
- ✅ 필터링 (전체/평가대기/평가완료)
- ✅ 평가 상태 실시간 표시

## 🎯 관리자 페이지 (index.html)

### UI 구조 - 3분할 레이아웃

- ✅ 평가자별 상세 점수 표시
- ✅ 평가자별 코멘트 표시

- 📋 평가자별 상세 점수
- 💬 평가자별 코멘트

## 🚀 사용 방법

### 1. 지원자 (apply.html)
```
1. 신규 지원 → 정보 입력
2. 💾 임시 저장 (선택)
3. 지원서 제출
4. 나중에 로그인하여 수정 가능 (평가 전)
```

### 2. 평가자 (evaluator.html)
```
1. 평가자 ID로 로그인
   - evaluator1 / eval123
   - evaluator2 / eval123
   - evaluator3 / eval123
2. 왼쪽에서 지원자 선택
3. 중간에서 지원서 확인
4. 오른쪽에서 평가 점수 입력
5. 평가 저장
6. 다음 지원자 평가
```

### 3. 관리자 (index.html)
```
1. admin / admin123 로그인
2. 지원자 선택
3. 중간 패널에서 평가 요약 확인
   - 평균 점수
   - 평가자 수
   - 평가자별 상세 점수
4. 오른쪽에서 상세 평가 내역 확인
   - 항목별 평균
   - 평가자별 코멘트
```
